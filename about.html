About

The goal of this website is to track the rapid growth of the lightning network. It tries to be most complete repository of lightning merchants, wallets, donation pages and services.

This is only possible thanks to the large number of contributors to the git repository. If you know a store/wallet/service/donation page not listed here, there are several ways to get it listed:

  • The fastest way is to make a pull request to the git repository. If it's a new store, add the information to sites.json; if it's a new testnet store, add it testnetSites.json; a new wallet, to wallets.json; a new donation page, to donationAddresses.json.
  • Open an issue in github
  • Send an email to lightningnetworkstores@gmail.com

For other lightning network resources such as explorers, learning materials and tutorials on setting a lightning node, check LNroute or Lopp.net
